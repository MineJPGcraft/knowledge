# MCJPG 组织规章制度

## 📋 组织简介

**MCJPG** 服务器交流组织（以下简称"本组织"）是由 **随风潜入夜**、**FireGuo**、**二氧化钛**、**Weasd-QAQ** 等服务器服主为了共同发展而联合起来的有组织的整体。

:::warning 重要提示
加入组织，即视为同意以下免责声明及全部规章制度。
:::

---

## ⚖️ 免责声明

1. **组织宗旨**  
   本组织旨在对各个有需求的服务器提供相对应的帮助扶持、宣传推广及技术支持等。

2. **建议参考性**  
   本组织提出的建议不具有专业性，仅供参考。遵循本组织提供的建议时，如造成不良后果，本组织不承担任何责任。

3. **活动自愿性**  
   在组织举行众筹活动时，各服服主可以自行决定是否参与，除特殊情况外，此类活动不具有强制性。

4. **自愿加入与退出**  
   本组织不强迫各服务器服主加入及退出，加入本组织者视为自愿加入，退出者视为自愿退出。本组织收集的各服地址、端口等材料仅作为审核、添加状态监测及提供给适合的玩家游玩所用，不会有其他的、非法的目的（对于向玩家提供的服务器相关信息，本组织仅起到传达作用）。

5. **退出处理**  
   在退出本组织时，除特殊情况外（如违反本组织相关规定），均视为自愿退出。

6. **域名回收政策**  
   退出本组织后，本组织将删除我方为退出服务器提供的如 `servernamexxx.mcjpg.dev` 域名，所造成的后果由退出者自行承担。

7. **最终解释权**  
   本组织保留对本免责声明的最终解释权及随时进行编辑的权利。如您对本免责声明有任何异议，请您及时私信联系本组织管理组。

:::danger 特别声明
**加入本组织者，视为已经阅读完毕并同意本免责声明。**
:::

## 🚪 加入本组织

### 📝 参与本组织活动

1. **审核流程**  
   进入本组织者，需要接受社区群组内审核员的审查，如不同意审查，即视作放弃加入本组织。

2. **遵守规定**  
   进入本组织后，应遵守相关规章制度并积极履行相对应的义务，违反规定者，将面临不同程度的处罚，严重者逐出本组织，永不允许加入。

3. **成员认定**  
   服主成功加入本组织服主群即视其服务器为本组织成员服。

---

### 📢 宣传条例

1. **标识要求**  
   成员服必须在 MCJPG 宣传网页中注明组织，具有正确的指向链接 [https://mcjpg.org/](https://mcjpg.org/)  
   （若无宣传网页请忽略）

2. **退出反馈**  
   为了组织良性发展，我们需要收集足够的信息，发现并解决存在的问题。因此建议各个成员服退出组织时需要提出退出组织的原因。

3. **活动支持**  
   组织允许并支持各服务器以集宣组织名义开展活动，组织允许并支持各服务器自由的、自主的组织活动（活动包括但不限于宣传活动、评比活动等）。

4. **服务商宣传免责声明**  
   本宣传公告旨在提供有关服务商宣传的一般信息，不构成任何形式专业建议，仅供参考。本组织为各成员提供更有效果的服务器解决方案，但不对任何因使用或依赖这些信息而产生的直接或间接损失承担责任。且本组织管理没有与任何服务商签订任何的利益分配协议。本组织保留对本免责声明的最终解释权。  
   如果您对此有异议，请及时向本组织负责人私信反馈。

## 🎮 如何在组织官网中添加自己的服务器？

### 步骤一：准备服务器信息

请提供以下必要信息：

| 信息项 | 说明 | 是否必需 |
|--------|------|----------|
| 服务器名称 | 您的服务器名称 | ✅ 必需 |
| 服务器图标 | 服务器的标识图标 | ⭕ 可选 |
| 服务器信息 | 服务器的基本信息（如IP地址、端口等） | ✅ 必需 |
| 服务器描述 | 服务器的详细介绍 | ✅ 必需 |
| 服务器版本 | 支持的Minecraft版本 | ✅ 必需 |
| 宣传链接 | QQ群链接、官网链接或宣传视频链接 | ✅ 必需 |

---

### 步骤二：提交申请

向负责人发送包含以下格式的申请信息：

```text
服务器名称：
服务器图标：
服务器信息：
服务器描述：
服务器版本：
宣传链接：
```

---

### 步骤三：等待审核

审核通过后，您的服务器将出现在 MCJPG 官网的服务器列表中。

:::tip 💡 高级选项
如果您有 GitHub 账号，可以直接按照服务器列表仓库的说明提交 Pull Request，帮助我们更快速地更新服务器列表！

🔗 服务器列表的 GitHub 仓库：[https://github.com/MineJPGcraft/Serverlist](https://github.com/MineJPGcraft/Serverlist)
:::
